# QuickReaderQR

QuickReaderQR is a simple and fast application that scans QR codes on Android and iOS devices.

The application also features a history page that stores all previously scanned codes for future reference.

Data is saved locally using an SQLite database, allowing users to keep their information on their phone without sharing it with any servers.

It also includes a page for generating QR codes.
